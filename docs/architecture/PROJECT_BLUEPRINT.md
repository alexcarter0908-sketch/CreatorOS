# CreatorOS Project Blueprint

## Vision
CreatorOS is an AI-powered content automation platform that helps creators generate, manage, and publish content from one dashboard.

## Phase 1 (MVP)

- Dashboard
- Project Management
- Script Generator
- Thumbnail Generator
- Video Prompt Generator
- SEO Generator
- Settings

## Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- Python

### Database
- PostgreSQL

## Development Rules

- Clean Architecture
- Modular Code
- Git Version Control
- Documentation First
- Testing Before Deployment
